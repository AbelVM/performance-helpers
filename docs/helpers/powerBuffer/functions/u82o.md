# Function: u82o()

> **u82o**(`buf`): `any`

Decode a UTF-8 encoded binary (Uint8Array / ArrayBuffer / Buffer) into a JS value by parsing JSON.

## Parameters

### buf

`any`

Binary input containing JSON UTF-8.

## Returns

`any`

Parsed JavaScript value.

## Throws

If the input type is not supported.

## Example

```ts
const obj = u82o(u8)
```
